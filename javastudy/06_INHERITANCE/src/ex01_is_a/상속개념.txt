 상속
 
 1. inheritance
 2. 어떤 클래스의 기능(메소드)을 다른 클래스에게 상속해 주는 것이다.
 3. 부모 클래스의 기능을 자식 클래스에게 상속해 주는 것이다.
 4. 부모 클래스 : 슈퍼 클래스(super, sup)
 5. 자식 클래스 : 서브 클래스(sub)
 6. 상속이 가능한 관계
	1) IS-A  관계 : 자식 IS A 부모. 연필은 학용품이다. 연필 IS A 학용품. TV는 전자제품이다.
	2) HAS-A 관계 : 자식 HAS A 부모. 철수는 연필을 가지고 있다. 철수 HAS A 연필. 군인은 총을 가지고 있다.
 7. 형식
 	class 부모클래스 {
 	
 	}
 	class 자식클래스 extends 부모클래스 {
 		자식 클래스에서 부모 클래스의 기능들을 사용할 수 있다.
 	}
 8. 다중상속은 불가능하다. 한 자식에 한 부모. A와 B는 상속관계가 없고, A와C B와C가 상속 관계일 때, A와 B에 Z라는 기능이 있을 때, C는 어디의 Z기능인지 모름.
 	모호성
 
 
 class Gun {
	 shoot() - 메소드
 }
 
 class Soldier {
 	Gun gun;
 	gun.shoot();
 }
