컬렉션 프레임워크

1. 각종 자료구조를 구현해 놓은 구현체이다.
2. Collection 인터페이스와 Map 인터페이스를 구현한 클래스이다.
3. Collection 인터페이스 구현체 ( List 인터페이스와 Set 인터페이스는 Collection 인터페이스의 자식임)
	1) List 인터페이스
		(1) ArrayList 클래스 (배열리스트 클래스)
		(2) LinkedList 클래스
		...
	2) Set 인터페이스
		(1) HashSet 클래스
		(2) LinkedHashSet 클래스 (순서대로 유지)
		...
4. Map 인터페이스 구현체       
		(1) HashMap 클래스 
		(2) TreeMap 클래스
		...
Map은 인스턴스의 대용품이다.
여기서 map은 mapping이다. 매핑은 사전처럼, 사과를 찾았을 때 그 뜻을 알려주는 식이다. A를 찾았을 때 그에 해당하는 값 알려주기.
변수와 저장된 값이 있다고 할 때, mapping을 하면 하나의 데이터로 여러 변수를 관리할 수 있게 해준다.
