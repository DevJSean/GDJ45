Writer 클래스

1. 문자 기반의 출력 스트림 처리 클래스이다.
2. 문자를 보낼 때 사용한다. 쉽게 말하면 채팅
3. FileWriter, BufferdWriter, OutputStreamWriter 등 Writer 클래스를
	상속 받은 클래스들을 사용한다.
	
4. 주요 클래스
	1) FileWriter
		(1) 문자를 파일로 보내는 클래스
		(2) 텍스트 파일을 만들 때 사용
		데이터를 그냥 보냄
	2) BufferedWriter
		(1) 보조 스트림(메인 스트림과 함께 사용)
		(2) 메인 스트림에 버퍼(임시기억장소)를 제공
		(3) 속도가 향상됨
		데이터를 그릇(버퍼)에 담아서 보냄
	3) OutputStreamWriter
		(1) 바이트 기반 출력 스트림을 문자 기반 출력 스트림으로 변경할 때 사용
		(2) 바이트 기반에서는 문자(특히 한글)가 깨질 수 있으므로 변환이 필요
